# UI Spec

## Principles

- Native SwiftUI.
- Apple system typography.
- Light and dark mode support.
- No third-party logos.
- No copied third-party app interface.
- Dense but readable information layout.
- Every primary screen displays: `仅供信息展示，不构成投资建议。`

## Tabs

1. Dashboard
2. Watchlist
3. Events
4. Reports
5. Settings

## Dashboard

Contains:

- Horizontal market index strip.
- Today market overview card.
- Watchlist quick list.
- Major event timeline.
- Mock data warning.
- Investment risk disclaimer.

## Event Card

Contains:

- Title.
- Summary.
- Affected symbols.
- Importance score.
- Event type.
- Publish time.
- Sentiment.
- Mock data label.

## Required Mock Warning

`当前为 Mock 演示数据，不代表真实市场价格。`

## Required Disclaimer

`仅供信息展示，不构成投资建议。`
