# Market Radar iOS Prototype

Static SwiftUI source prototype for Phase 1.

## Scope

- Dashboard, Watchlist, Events, Reports, and Settings screens.
- Local Mock data only.
- No real backend connection.
- No third-party Swift packages.
- No Apple signing, certificates, provisioning profiles, APNs, or Keychain access.

## Disclaimer

The UI must display:

`当前为 Mock 演示数据，不代表真实市场价格。仅供信息展示，不构成投资建议。`
